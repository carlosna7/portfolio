theMov

O theMov é o meu segundo projeto em React, para seu desenvolvimento tive como prioridade estudar e aplicar as funcionalidades dos frameworks Next.js e Tailwind CSS.

Além de utilizar ambos os frameworks, neste projeto, fiz uso da API do The Movie DataBase, que consiste em uma lista de filmes. Os usuários podem pesquisar ou obter mais informações sobre os filmes em destaque.
