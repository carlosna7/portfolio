Clone Trello

O projeto em que mais utilizei o JavaScript foi o CardUIT. Nele, utilizei diversas funcionalidades que aprendi ao longo do tempo para desenvolver um projeto baseado no site Trello, que é um aplicativo de gerenciamento de projetos.

Para seu desenvolvimento, foi utilizado além de HTML e CSS o JavaScript como linguagem padrão, tanto para a criação dos elementos HTML pelo JavaScript quanto para adicionar a funcionalidade de arrastar e soltar as tarefas nos cards, mudar o tema, fazer a edição das tarefas e adicionar comentários.
