Custos

O Custos foi o meu primeiro projeto desenvolvido com a biblioteca React. Essa aplicação tem como objetivo simular um sistema de criação de projetos, onde cada projeto criado terá um valor máximo de gastos definido na criação, e esse valor máximo de gastos diminui de acordo com os serviço que são adicionados.

Para desenvolver o Custos, foram utilizadas as tecnologias da biblioteca React, JavaScript, CSS e uma Fake API chamada JSON Server. Para acessar a API, as instruções estão no README do repositório.
