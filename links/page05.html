Clima agora

Para aprimorar minha prática no consumo de dados de APIs, decidi criar o site Clima Agora. Para o seu desenvolvimento, utilizei apenas HTML, CSS e Javascript.

Neste projeto, utilizei a API Weatherbit para obter dados climáticos e previsões do tempo. Além disso, a segunda API utilizada foi a Flagsapi para associar as bandeiras aos países pesquisados.
